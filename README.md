# MoodMap

## Overview

MoodMap is a Firebase + Next.js project that allows users to visualize moods on a map. It integrates with Mapbox GL for a rich mapping experience.

## Getting Started

Follow these instructions to set up the project and run it locally.

## Prerequisites
- Node.js
- npm
- Firebase account with necessary permissions

## Installation

1. Clone the repository.
2. Install dependencies using `npm install`.

## Usage

To start the development server, run:
```
npm run dev
```
